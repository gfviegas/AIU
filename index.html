Assync Image Uploader

This jQuery + AJAX + PHP plugin uploads an image asynchronous to your FTP server and just load it in your page. That's right! No reloads!! You can send and show your image without refresh!!

To instructions, suggestions and more about the AIU, check the "Read Me" file in Git-Hub.
The code-identation, tags and names are in portuguese.
